## 1. 产品概述

### 1.1 产品定位

快速、简洁的在线图片背景去除工具。用户上传图片，自动去除背景并支持下载透明背景 PNG。

### 1.2 核心价值主张

- **快**：5秒内完成背景去除
- **简**：无需注册、上传即用、无广告
- **净**：不存储用户图片、处理完即释放内存

### 1.3 目标用户

- 电商卖家（制作白底产品图）
- 设计师（快速提取素材）
- 普通用户（证件照换底色等日常需求）

## 2. 技术架构

### 2.1 技术选型

| 层级 | 技术方案 |
|------|---------|
| 前端 | Cloudflare Pages（静态托管）|
| 后端 | Cloudflare Worker（图片中转，无持久化）|
| 图像处理 | Remove.bg API |
| 域名 | 自有域名解析至 CF |

### 2.2 数据流向

```
用户浏览器
    │
    │ ① 图片上传（multipart/form-data）
    ▼
Cloudflare Worker
    │
    │ ② 转发（内存流转，不落盘）
    ▼
Remove.bg API
    │
    │ ③ 返回透明背景 PNG
    ▼
Worker → 用户浏览器
    │
    │ ④ 直接输出图片流
    ▼
处理完成，内存释放
```

### 2.3 环境变量

| 变量名 | 说明 | 来源 |
|--------|------|------|
| `REMOVE_BG_API_KEY` | Remove.bg API Key | 用户自行申请 |

## 3. 功能需求

### 3.1 核心功能（必须）

#### F1 - 图片上传
- 支持拖拽上传和点击上传
- 支持格式：JPG、PNG、WebP
- 单文件大小限制：10MB
- 前端预览上传图片

#### F2 - 背景去除处理
- 调用 Remove.bg API 完成处理
- 显示处理中状态（加载动画）
- 超时时间：30秒
- 失败时显示友好错误提示

#### F3 - 结果展示与下载
- 展示去背后的图片预览
- 一键下载为 PNG（透明背景）
- 支持"重新上传"继续处理下一张

### 3.2 边界处理

| 场景 | 处理方式 |
|------|---------|
| 图片过大（>10MB） | 前端拦截，提示文件过大 |
| 格式不支持 | 前端拦截，提示支持格式 |
| API 调用失败 | 展示错误信息，支持重试 |
| API 超时（>30s） | 超时提示，可重试 |
| 并发处理中 | 禁用上传入口，排队处理 |

## 4. 非功能需求

### 4.1 性能

- 首屏加载 < 2秒（静态资源 CDN 加速）
- 图片处理时间取决于网络和图片大小（Remove.bg 通常 3-10秒）
- Worker 响应时间 < 50ms（中转开销）

### 4.2 安全

- 不存储任何用户图片
- API Key 存储于 Worker 环境变量，不暴露于前端
- 前端对文件类型和大小做双重校验

### 4.3 可用性

- Cloudflare 全球 CDN 保障访问稳定性
- Remove.bg API SLA 保障

## 6. 部署清单

### 6.1 前置依赖

- [ ] 注册 Cloudflare 账号
- [ ] 申请 Remove.bg API Key（[官网](https://www.remove.bg/api)）
- [ ] 准备域名（可选）

### 6.2 部署步骤

1. **创建 Worker 项目**
   ```bash
   npm create cloudflare@latest
   # 选择 Worker 项目类型
   ```

2. **配置环境变量**
   - 在 Cloudflare Worker 设置中添加 `REMOVE_BG_API_KEY`

3. **编写 Worker 代码**
   - 实现图片中转逻辑

4. **部署 Worker**
   ```bash
   wrangler deploy
   ```

5. **创建 Pages 项目**
   - 关联 Git 仓库或直接上传
   - 配置自定义域名（可选）

6. **配置 Pages 环境变量**
   - `VITE_API_ENDPOINT`: Worker 访问地址
